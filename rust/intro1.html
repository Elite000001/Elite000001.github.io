  • Step 1: What is Rust?

    Before we write our first program in Rust, it’s best to understand why this language was created.

    A toy project

    Rust started as a toy project by Grayden Hoare while he was still working as an engineer with Mozilla, the company known for the Firefox web browser. He wanted a robust language, avoiding the usual mistakes created by C and C++ developers by building a language written in OCaml in the year 2006. Mozilla saw the potential of his work and built Rust around Servo, an experimental web engine which later was abandoned by Mozilla in the year 2020. Servo is still maintained and is now under the stewardship of the Linux Foundation.

    Main features

    Rust addresses memory-safety issues by adopting a unique approach to memory management. It uses a system of ownership with a set of rules that the compiler checks at compile time. No garbage collector is needed, and no runtime overhead is incurred. The ownership system helps manage memory and thread safety and eliminates common errors such as null pointer dereferencing, double free error, and data races, thereby greatly reducing security vulnerabilities associated with these issues.

    Empowering

    Even before Mozilla abandoned Servo, Rust has gained adoption by various parts of the industry since 2015. Rust has created an ecosystem of sustainable software and thus, a foundation was formed called the Rust Foundation whose members are from large enterprises such as Google, Meta, Amazon and Microsoft.

    Community

    Rust has built a large fanbase over the years. More and more developers are learning Rust because of what it offers to allow them to write mission-critical software.

    Not surprising for a language that has gained popularity and the most admired language based on the 2023 StackOverflow Survey and it has done this since 2016! 🥳

    Oh and say hi to Ferris, our cute little crab mascot! 👋

  • Step 2: Why should you learn Rust?

    Memory Safety

    An estimate of seventy percent (70%) of security vulnerabilities in computing were found to be mostly memory-safety issues. Google believes that a “Secure by Design” approach towards these issues will work towards a better solution — investing in memory-safe languages, especially the Rust programming language.

    Google and Microsoft are confidently investing in Rust, driven by its unwavering commitment to safety and performance. Rust's robust system eliminates common programming pitfalls like null pointer dereferencing and data races, effectively safeguarding against system crashes and security vulnerabilities.

    This positions Rust as the go-to language for constructing dependable system-level software. Adding to its appeal, Rust boasts a modern and expressive syntax, streamlining code writing and maintenance. Its vibrant, active community is a testament to Rust's ongoing evolution and enhancement.

    Accelerating Adoption

    Companies have been accelerating their adoption of Rust in their tech stack. Meta, formerly known as Facebook, has been adopting Rust for its web and mobile services. CloudFare has open-sourced its Rust framework for building reliable network services called pingora! ❤️‍🔥

    Among other things, the Linux kernel has experimented to integrate Rust code and also has a guide on how to contribute Rust to the Linux kernel.

    Adoption of this language keeps on accelerating in various parts of the industry such as cybersecurity, operating systems development, automation, game development, web technologies, blockchain and Web3 🔥

    Developer Opportunities

    It may not be evident now but as more and more of the industry are looking towards Rust adoption, it became clear that we should also learn this language.

    Github, one of the world’s largest developer platforms, also has an informative article about why Rust is the most admired language among developers in web development, operating systems development, and even blockchain and Web3 development.

    Rust has brought more opportunities to allow developers to explore new heights, innovate the field, and also empower themselves by leveraging Rust across the industry.

    If you still have doubts about learning Rust, then keep on reading and learning. You will appreciate how Rust empowers developers like you.
