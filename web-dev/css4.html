Laying Elements Out with CSS Grid

5 steps

In this tutorial, you will be learning about the CSS Grid module that lets you lay content out in a two-dimensional layout system with rows and columns. With Grid, you can build complex, modern layouts that are responsive.

Join us on Discord

Learning Outcomes

By the end of this tutorial, you will be able to:

  1. List at least 5 CSS Grid properties
  2. Explain the best use cases for CSS Grid
  3. Develop web page layouts using CSS Grid properties and values

Tutorial Steps

Total steps: 5

  • Step 1: Discovering CSS Grid

    CSS Grid is the very first CSS module created specifically in 2011 to solve layout problems. It helps to create layouts in which our elements won't jump around or change width as we move from page to page. Since it is a two-dimensional layout system for the web, you will see that most of the properties will have both row and column directions.

    In this step, we will be reading up about CSS Grid. If you plan on working on your web development skills, it would be great if you made your own cheat sheet or notes for CSS Grid.
