JavaScript Classes

6 steps

In this tutorial, we will learn everything there is to know about Classes. In JavaScript, a class is a blueprint for creating objects with similar properties and behaviors. It serves as a template or a structure that defines the characteristics and functionalities of objects that belong to it.

Like a function, when writing a class, there are 2 steps to it. First, it is to create the structure for the class and next is to create an object with a class. Think of it like baking a cake. Before baking a cake, you might probably have to look for a recipe or even create one yourself. Once you have the recipe, it is time to use the recipe to bake your cake. These are essentially the 2 steps to creating a class. We will be exploring how to create a class and to also explore what instances and inheritance are as well. Without further ado, lets start coding!

Tutorial Steps

Total steps: 6

  • Step 1: Classes and Constructors

    In JavaScript, a class is a blueprint for creating objects that share the same properties and methods. It is a way to define a new type of object with its own characteristics and behaviors. The class serves as a template or prototype from which individual objects, known as instances, can be created.

    This is how a typical class will look in JavaScript. 

    class MyClass {
 constructor(property1, property2) {
  this.property1 = property1;
  this.property2 = property2;
 }

 method1() {
  // Method implementation
 }

 method2() {
  // Method implementation
 }
}
    JavaScript

    In the above example, we define a class named MyClass using the class keyword. Inside the class, we can define properties and methods.

    The constructor method is a special method that gets called when an instance of the class is created using the new keyword. It is used to initialize the object's properties. In the example, property1 and property2 are the parameters of the constructor, and the values passed to these parameters are assigned to the respective properties of the instance.

    The method1 and method2 are regular methods of the class. They define the behaviour or actions that instances of the class can perform. You can add any number of methods to a class.

    To create an instance of the class and use its properties and methods, you can do the following: 

    const myInstance = new MyClass("value1", "value2");
console.log(myInstance.property1); // Output: "value1"
myInstance.method1(); // Call to method1
    JavaScript

    In the above code, we create a new instance of MyClass using the new keyword and pass values for the properties property1 and property2. We can access and modify the properties of the instance using dot notation (myInstance.property1).

    Note that JavaScript classes were introduced in ECMAScript 2015 (ES6) and are primarily syntactic sugar over the existing prototype-based inheritance model in JavaScript. Underneath the class syntax, JavaScript still uses prototypes to achieve inheritance and object creation.

  • Step 2: Our First Class in JavaScript

    Now, we will create our first class in JavaScript. Create a new file called index.js . We will be creating a class that will store the properties of a book. Add the following code as shown below into the index.js file

    class Book {
  constructor(title, author, ISBN, availability = true) {
    this.title = title;
    this.author = author;
    this.ISBN = ISBN;
    this.availability = availability;
  }
}
    JavaScript

    In the above code, we define a class named Book using the class keyword (line 1). The class has a constructor method, which is a special method that gets called when an instance of the class is created. The constructor method takes parameters such as title, author, ISBN, and availability (with a default value of true). Inside the constructor (line 2), we assign these parameter values to the respective properties of the instance using the this keyword (lines 3 to line 6).

    *ISBN is the International Standard Book Number. Every book has an ISBN.

  • Step 3: The Library Class

    Next, we will create a class called Library. In this class, we will add methods. Methods in classes are functions that are defined within the scope of a class. They define the behaviours or actions that instances of the class can perform. Methods allow us to encapsulate related functionality within a class, making the code more organized and modular.

    In the context of a class, methods are defined as regular functions, but they are attached to the class and can be accessed and invoked on instances of the class. Methods can interact with the properties of the class and perform operations based on those properties.

    In the same index.js file, add the following code below. 

    class Library {
  constructor() {
    this.books = [];
  }

  addBook(book) {
    this.books.push(book);
  }

  removeBook(ISBN) {
    this.books = this.books.filter((book) => book.ISBN !== ISBN);
  }

  displayAvailableBooks() {
    console.log("Available Books:");
    this.books.forEach((book, index) => {
      if (book.availability) {
        console.log(`${index+1}. ${book.title} by ${book.author}`);
      }
    });
  }
}
    JavaScript

    In the code above, we added the functions of Adding a Book, Removing a Book, and displaying all the books in the Library. These functions within the class are called methods.

    You may notice that in classes, all variables, also known as properties, will have a this at the front. For example this.title . The word this is referring to the class. Doing so will allow us to access the classes variables.

    To add a book, you may simply push the Book object into the array. To remove a book, we can use the .filter() method to remove the Book object based on the ISBN. Since ISBN is a unique number, we can safely remove the Book object just by defining the ISBN. Lastly, to display all the books available, we will use the .forEach() method to iterate through the this.book array.

    At this point, your code should look like this.

  • Step 4: Class Inheritance

    We will now create another class. But this class will take on similar properties to the Book class. This is also known as Inheritance.

    Inheritance in the context of a class refers to the ability of a class to inherit properties and methods from another class, known as the parent or base class. The class that inherits from the parent class is called the child or derived class. Inheritance allows the child class to reuse and extend the functionalities and characteristics of the parent class, promoting code reuse and maintaining a hierarchical structure.

    Inheritance is a fundamental concept in object-oriented programming (OOP) that helps to organize and model relationships between classes. It enables the child class to inherit the properties and methods of the parent class, reducing code duplication and promoting modularity.

    The child class can access and use the inherited properties and methods of the parent class as if they were defined within the child class itself. Additionally, the child class can override inherited methods to provide specialized implementations or introduce new properties and methods specific to the child class.

    Add the following code as shown below to the end of the file. 

    class ReferenceBook extends Book {
  constructor(title, author, ISBN, category, availability = true) {
    super(title, author, ISBN, availability);
    this.category = category;
  }
}
    JavaScript

    In the above code, we created a new class called ReferenceBook that extends the Book class using the extends keyword. The ReferenceBook class inherits the properties and methods of the Book class. It also introduces a new property called category. When creating a ReferenceBook instance, we pass the additional category argument to the constructor of the ReferenceBook class and call the super() method (line 35) to invoke the parent class constructor. This way, ReferenceBook will not only have the new property, category, but the new Class will also inherit the properties of Book as well.

  • Step 5: Creating Object Instances

    Once we have written our three classes, we will now create instances of those classes. You may think of this as creating clones or creating copies of these objects.

    Now, add the following code to the end of index.js.

    // Create instances of the Library, Book, and ReferenceBook classes
const library = new Library();

// Add books to the library
const book1 = new Book("The Great Gatsby", "F. Scott Fitzgerald", "9780743273565");
const book2 = new Book("To Kill a Mockingbird", "Harper Lee", "9780060935467");
const book3 = new ReferenceBook("1984", "George Orwell", "9780451524935", "Dystopian Fiction");

library.addBook(book1);
library.addBook(book2);
library.addBook(book3);

// Display available books
library.displayAvailableBooks();

console.log("\nRemoving book...");

// Remove a book from the library
library.removeBook("9780743273565");

// Display available books again
library.displayAvailableBooks();
    JavaScript

    In line 41, we created an instance of the Library class using the new keyword and assigned it to the variable library. This creates an empty library object with an initially empty list of books.

    In lines 44 and 45, we created two instances of the Book class, representing different books in our library. The books are initialized with respective properties such as title, author, and ISBN. We assign each book instance to variables book1 and book2.

    Then, in line 46, we created an instance of the ReferenceBook class called book3. The ReferenceBook class extends the Book class and has an additional property called category. We pass the title, author, ISBN, and category as arguments to the constructor.

    In lines 48 to 50, we add the book instances to the library using the addBook() method of the Library class. We pass each book object as an argument to the method, which adds the book to the library's collection of books.

    In line 53, the displayAvailableBooks() method of the Library class outputs the list of available books to the console. This method iterates through the books in the library and displays the title and author of each available book.

    In line 58, we will call the removeBook() method of the Library class, passing the ISBN of the book we want to remove as an argument. This method removes the book from the library's collection based on the ISBN.

    Finally in line 61, we will call the displayAvailableBooks() method again to show the updated list of available books after removing a book.

Once we have identified where our file is located, we can run the file by using the node command. Below is an example of how you can navigate and run your code.
