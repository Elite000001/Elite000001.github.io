How Flexible Can You Be With Flexbox?

3 steps

In this tutorial, you will learn more about the CSS Flexible Box Layout module, also known as Flexbox, which is commonly used in layouts. There are various things you can do with Flexbox properties, such as altering a containers width or how you distribute your items within a container. As its name suggests, you will have plenty of flexibility in arranging items. You just need to learn the properties and values for your CSS ruleset!

Tutorial Steps

Total steps: 3

  • Step 1: What is Flexbox?

    Flexbox is a pretty powerful property that you can use to flexibly arrange elements on your web page. Because flexbox can align items and distribute space in a container even though the size of the container is unknown, it is very useful for responsive design. This means that items can shrink or grow depending on the screen size. Also, for web developers who have had issues with centering elements, flexbox allows for the centering of elements both horizontally and vertically with just the display, justify-content and align-items properties.
